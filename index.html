TodoList

Il s'agit d'une application de gestion de tâches développée avec React et Material UI, permettant d'afficher toutes les tâches, de filtrer les tâches terminées ou non terminées, d'ajouter de nouvelles tâches via un formulaire intuitif, de consulter les détails de chaque tâche, de les modifier ou de les supprimer, et de marquer les tâches comme terminées en cochant une case. Page de Formulaire Interactif avec React

Une page web interactive développée avec React pour la validation des données. La page contient un formulaire où l’utilisateur saisit son nom, âge, numéro de téléphone, statut professionnel et salaire. Les règles de validation sont : L’âge doit être compris entre 18 et 100 Le numéro de téléphone doit contenir 10, 11 ou 12 chiffres Lors de la soumission du formulaire, un message s’affiche pour confirmer le succès ou avertir l’utilisateur si les données sont incorrectes. Le projet met l’accent sur l’interaction en temps réel et la validation des entrées.
